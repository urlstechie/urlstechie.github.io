urlstechie was launched in March 2020. The project started as a small GitHub action built by @SuperKogito for a side project and grew from there to its current state. Project aims to automate links testing and simplify it as it a crucial part of continuous integration. Project is not limited to online environments as we also provide a python-package for offline testing. The current developers and maintainers are:

  • Ayoub Malek aka @SuperKogito

  • Vanessa Saurus aka @vsoch
